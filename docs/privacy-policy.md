# Privacy Policy — Smart Lorem Ipsum Generator

**Last updated:** March 2026

## Data Collection
Smart Lorem Ipsum Generator does **not** collect, store, or transmit any personal data.

## Permissions Used
- **clipboardWrite** — to copy generated text to your clipboard
- **storage** — to save your preferences (theme, presets) via Chrome Sync
- **activeTab** — to insert text into the active page when using context menu or keyboard shortcut
- **contextMenus** — to provide right-click menu options
- **scripting** — to inject text into editable fields on the active tab

## Network Requests
This extension makes **zero** network requests. It works entirely offline with no external API calls.

## Analytics & Tracking
No analytics, telemetry, or tracking of any kind.

## Third-Party Services
No third-party services, SDKs, or libraries are used.

## Data Storage
User preferences (theme, count, presets) are stored locally via `chrome.storage.sync` and synced across your Chrome browsers through your Google account. No data is sent to the extension developer.

## Contact
For questions about this privacy policy, open an issue on the GitHub repository.
